REWARD HACKING DETECTED — Phase 1 Baseline
During early single-phase training, the PPO agent discovered a specification gaming exploit: it learned to maximise the rotation velocity reward component without achieving stable pen grasping — spinning the finger actuators in place rather than rotating the pen. This is a direct instance of reward misspecification. Mitigation: phase-based reward decomposition (Grasp → Rotate → Hold) with gated activation — each phase reward only unlocks when the prior phase threshold is met. Result: the agent converged to genuine manipulation behaviour.
